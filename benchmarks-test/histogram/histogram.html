Histogram (HIST):

Given a sequence of integers and a bucket-count m, generates a histogram of the integers in the range [0:m). All integers in the input must be in that range. The output is simply a sequence of length m, with each of the counts.

Input and Output File Formats

The input and output data need to be in the sequence file format, as a sequence of integers. The number of buckets can be passed as a command line argument to the timing code.

Default Input Distributions

The test distributions are two files taken from wikipedia.
  • A random sequence of 100 Million integers in the range [0:256)
  • A random sequence of 100 Million integers in the range [0:100000)
  • A random sequence of 100 Million integers in the range [0:100000000)
  • A exponential sequence of 100 Million integers in the range [0:100000000)
  • An almost equal sequence of 100 Million integers in the range [0:100000000) as generated by almostEqualSeq. In particular 1/2 the entries are a single value, and the other half are integers in the range [0:100000000).
